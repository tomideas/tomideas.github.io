引用頁面內容

將當前瀏覽的網頁內容擷取為 AI 對話的上下文。

使用方式

點擊 📄 按鈕 → 系統擷取頁面 → 顯示預覽卡片 → 可展開檢視或取消引用。

擷取模式

於設定 → 引用頁面可選擇擷取模式：

Full Page（全文模式）

將完整 HTML 轉為純文字後送入對話。

Markdown Smart Extraction（智慧精讀模式）

使用 Mozilla Readability 與 Turndown 轉為 Markdown，利於結構化閱讀。

💡 推薦一般閱讀與長文摘要建議使用此模式，可減少雜訊並保留標題與段落結構。

自訂 CSS 選擇器模式

以 CSS 選擇器指定要包含或排除的區塊，只擷取你需要的內容。

字元限制

單次引用約為 1,000～200,000 字元（依設定與頁面而定）。

⚠️ 過長上下文費用內容越長，佔用的上下文與 API 費用可能越高，請依需求調整擷取範圍或模式。

智慧捲動

支援虛擬捲動與延遲載入偵測，盡量將可見或可展開的內容納入擷取範圍。

追蹤擷取狀態

系統會記錄 URL，避免在同一頁面重複擷取時產生多餘負擔。
