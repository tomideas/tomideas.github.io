聊天記錄

開啟歷史記錄

使用側邊欄頂部的歷史按鈕，滑出面板即可檢視與管理過往對話。

功能

  • 瀏覽對話：依時間排序瀏覽所有已儲存對話。
  • 重新命名：點擊對話標題可重新命名。
  • 選擇與批量操作：支援全選與批量刪除。
  • 匯出：將對話內容匯出為 JSON 備份。
  • 匯入：從 JSON 檔案匯入對話記錄。
  • 刪除：刪除單則或所選對話。

儲存機制

對話資料儲存在 chrome.storage.local，不會透過 Chrome 同步至其他裝置。

自動清理

系統會自動清理符合以下條件的對話以節省空間：

  • 對話總數超過 200 筆
  • 對話建立超過 90 天
  • 對話僅包含 1 則或更少的訊息
ℹ️ 空白對話空白對話不會被保存。
