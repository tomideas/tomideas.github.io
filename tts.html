語音朗讀 (TTS)

使用瀏覽器內建的 Web Speech API，將 AI 回覆以語音朗讀。

使用方式

點擊 🔊 朗讀按鈕開始朗讀；再次點擊同一按鈕可停止。

文字前處理

朗讀前會自動清理回覆內容：

  • 移除思考區塊（<think>、<thinking>、<thought>）
  • 移除 HTML 標籤和 Markdown 語法
  • 移除 emoji 和裝飾符號
  • 根據使用者語言偏好自動進行繁簡轉換

設定（設定 → 語音朗讀）

語音選擇

從系統與瀏覽器提供的可用語音清單中選擇朗讀聲音。

💡 更多語音在 macOS 或 Windows 的系統設定中可下載並安裝額外語音，安裝後通常會出現在此清單中。

語速（Rate）

調整朗讀速度。

音調（Pitch）

調整朗讀音調高低。

預覽

使用試聽按鈕可預先聽取目前設定效果。
