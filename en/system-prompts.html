System Prompts

Define the AI's role, behavior, and response style.

How to Use

Select a prompt from the dropdown at the top of the sidebar. Changes take effect immediately.

Built-in Prompts

Several presets are included, such as Default Assistant, Translation Assistant, Writing Assistant, and more — ready to use out of the box.

Manage Prompts (Settings → Prompts)

Add Prompt

Create a new custom system prompt.

Edit Prompt

Modify the content and name of an existing prompt.

Delete Prompt

Remove prompts you no longer need.

Set as Default

Use the radio button to designate which prompt is applied by default when starting a new conversation.

Show / Hide

Use the toggle to control whether a prompt appears in the sidebar dropdown.

💡 Declutter the Menu Hide infrequently used prompts to keep the dropdown clean and concise.

Reset Prompts

Restore built-in prompts and related provider settings to their defaults.

⚠️ Overwrites Customizations Resetting will overwrite your custom content and related settings. Make sure to back up if needed.
