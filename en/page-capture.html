Page Capture

Capture the content of the current webpage as context for your AI conversation.

How to Use

Click the 📄 button → the system captures the page → a preview card appears → you can expand to view or cancel the capture.

Capture Modes

Go to Settings → Page Capture to choose a capture mode:

Full Text Mode

Converts the full HTML into plain text and sends it to the conversation.

Reader Mode

Uses Mozilla Readability and Turndown to convert the page into Markdown, preserving structure for easier reading.

💡 Recommended For general reading and long-text summarization, Reader Mode is recommended — it reduces noise while preserving headings and paragraph structure.

Custom Mode

Use CSS selectors to specify which sections to include or exclude, capturing only the content you need.

Character Limit

A single capture is approximately 1,000–200,000 characters (depending on settings and the page).

⚠️ Long Context Costs The longer the content, the more context and API costs it may consume. Adjust the capture scope or mode as needed.

Smart Scrolling

Supports virtual scrolling and lazy-load detection, capturing as much visible or expandable content as possible.

Capture State Tracking

The system records the URL to avoid redundant captures on the same page.
