Installation

System Requirements

  • Google Chrome browser (version 116 or above, with Side Panel API support)
  • Or any Chromium-based browser (Edge, Brave, Arc, etc.)

Installation Steps

Method 1: Install from Source (Developer Mode)

  1. Download or clone the Momo AI extension source folder sider/
  2. Open Chrome and navigate to: chrome://extensions/
  3. Enable the Developer mode toggle in the top-right corner
  4. Click Load unpacked
  5. Select the sider/ folder
  6. Installation complete! You'll see the Momo AI icon in the toolbar

Method 2: Using Packaged .crx File

  1. Open chrome://extensions/
  2. Drag and drop the .crx file onto the extensions page
  3. Confirm the installation prompt

After Installation

Once installed, you can:

  • Click the toolbar icon to open the sidebar
  • Set up a keyboard shortcut — go to chrome://extensions/shortcuts to configure a custom shortcut for quick sidebar access (see Shortcuts page)
  • Configure your preferred AI provider and API key in the settings page
💡 First Time Use On first launch, Momo AI will automatically detect your browser language and set the interface language accordingly.
