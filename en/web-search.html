Web Search

Built-in web search lets AI query the latest information from the internet during conversations.

Enabling Search

Click the 🔍 search icon to toggle; when enabled, every message you send will automatically trigger a web search.

💡 Auto-Trigger Even when search is turned off, messages starting with "look up", "search", or "search for" will automatically trigger a web search.

OpenRouter Server-Side Search

When using OpenRouter, the :online feature is automatically enabled, powered by the Exa engine.

Search Modes

  • Quick Search: Summaries only, fast.
  • Off: Visits pages to retrieve content, slower but more accurate.

Search Settings

In settings you can adjust: search engine, number of results (1–20), quick search mode, auto-visit URLs, default search on, and test search.

Viewing Search Sources

Click the 🔍 button to view source titles, URLs, and summaries.
