Shortcuts

Global Shortcuts

Momo AI does not bind any global shortcuts by default. You need to set up your preferred key combination in Chrome settings.

Setting Up Shortcuts

  1. Type chrome://extensions/shortcuts in the Chrome address bar
  2. Find Momo:AI
  3. In the input field next to "Open sidebar", press the key combination you want to use (e.g. Cmd+Shift+Y or Alt+Shift+Y)
  4. Once set, you can use the shortcut to open the sidebar on any page
💡 Tip We recommend Cmd+Shift+Y (macOS) or Alt+Shift+Y (Windows/Linux) — easy to remember and unlikely to conflict with other apps.

Sidebar Shortcuts

Key Description
Enter Send message
Cmd/Ctrl+Enter Send message
Shift+Enter New line (without sending)
Escape Stop generation

Browser Shortcut Management

You can also modify extension shortcuts at chrome://extensions/shortcuts and set the scope: active only within Chrome windows, or global (triggers even when Chrome is not in the foreground, depending on your system and Chrome version).
