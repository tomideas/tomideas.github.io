Multi-language

Supported Languages

Language Code Note
Traditional Chinese hant Default
Simplified Chinese hans —
English en —

Switching Languages

Go to Settings → General → Language to select the interface language.

Auto Detection

On first use, the language is automatically detected based on your browser language: zh-TW, zh-HK → Traditional Chinese; zh-CN, zh → Simplified Chinese; all others → English.

Coverage

Translations cover the sidebar UI, settings page, action tooltips, error messages, and built-in system prompts — all user-visible strings.
