Quick Start

After installation, follow these steps to start using Momo AI in minutes.

Step 1: Open the Sidebar

There are three ways to open the Momo AI sidebar:

  1. Click the toolbar icon — Click the Momo AI icon in the Chrome toolbar
  2. Keyboard shortcut — You need to set this up first at chrome://extensions/shortcuts, see the Shortcuts page for details
  3. Float ball — If the float ball feature is enabled, simply click the floating button on the page

Step 2: Set Up AI Provider

  1. Click the ⚙️ Settings button in the top-right corner of the sidebar
  2. Select AI Models from the left navigation
  3. Choose the provider you want to use from the list (e.g. OpenAI, Google AI, DeepSeek)
  4. Enter the corresponding API key (some local providers like Ollama don't require a key)
  5. Click Test Connection to verify the configuration
💡 No API Key Needed? If you're using Ollama or LM Studio, just make sure the local service is running — no API key is required.

Step 3: Start Chatting

Go back to the sidebar, type your question in the input box at the bottom, then press Enter or click the send button.

You can also try clicking the suggested topics on the welcome screen to quickly start a conversation.
