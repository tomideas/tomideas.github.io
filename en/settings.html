Settings Overview

Opening Settings

You can access settings from the ⚙️ icon at the top of the sidebar, or by right-clicking the extension icon → Options to open the full settings page.

Settings Categories

General

Interface language, theme, font size (12–20), and font weight (200–700). For language and interface details, see Multi-language.

Sidebar

Whether to show the Float Ball, sidebar-related options, and Shortcuts hints.

AI Models

Provider, API endpoint, API key, model list, and connection testing. See AI Providers for details.

Web Search

Search engine, number of results, and advanced options. See Web Search for details.

Page Capture

Settings for capturing the current tab's content and sending it to the model. See Page Capture for details.

Text-to-Speech

TTS voice and playback behavior. See Text-to-Speech (TTS) for details.

Prompts

Customize and manage system prompts. See System Prompts for details.
