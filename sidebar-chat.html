側邊欄聊天

Momo AI 的核心功能是以 Chrome Side Panel API 打造的側邊欄聊天介面。

介面概覽

  • 頂部工具列：系統提示詞選擇器、聊天記錄、新對話、設定
  • 訊息區域：Markdown 渲染、思考過程摺疊、操作按鈕
  • 輸入區域：模型選擇、功能按鈕列（🖼️ 圖片上傳、📄 引用頁面、🔍 聯網搜尋）、文字輸入框、字元計數器、送出按鈕

對話操作

  • Enter：送出訊息
  • Cmd+Enter（macOS）：同樣送出訊息
  • Shift+Enter：換行（不送出）
  • Escape：停止生成

自動捲動

對話會自動跟隨最新內容；若你手動向上捲動檢視較早訊息，自動捲動會暫停。點擊 ⬇ 按鈕可恢復跟隨最新內容。

串流回覆

回覆以 SSE（Server-Sent Events）串流傳送。依回覆長度採用不同更新頻率：短回覆約每 80ms、中約 300ms、長約 800ms 更新一次，以平衡流暢度與效能。

歡迎畫面

尚未開始對話時會顯示歡迎畫面，並提供建議話題，可一鍵開始聊天。
