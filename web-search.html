聯網搜尋

內建聯網搜尋功能，讓 AI 在對話中能即時查詢網路最新資訊。

啟用搜尋

點擊 🔍 搜尋圖示開關；開啟後，每次發送訊息都會自動搜尋。

💡 自動觸發即使搜尋關閉，以「查一下」「搜尋」「search for」開頭的訊息也會自動觸發聯網搜尋。

搜尋引擎

引擎 費用 API 金鑰 說明
DuckDuckGo（預設） 免費 不需要 透過 HTML 擷取，無需註冊
Brave 免費額度 需要 高品質搜尋結果
Tavily 免費額度 需要 專為 AI 設計的搜尋 API

OpenRouter 伺服器端搜尋

使用 OpenRouter 時會自動啟用 :online 功能，由 Exa 引擎驅動。

搜尋模式

  • 簡易搜尋：僅摘要，快速。
  • 關閉：造訪頁面取得內容，較慢但較精確。

搜尋設定

於設定中可調整：搜尋引擎、結果數量（1–20）、簡易搜尋模式、自動造訪 URL、預設開啟搜尋，以及測試搜尋。

搜尋來源查看

點擊 🔍 按鈕可查看來源的標題、URL 與摘要。
